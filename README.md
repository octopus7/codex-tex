# Codex Tex

Electron + Vite + React + Three.js 기반 OBJ 텍스처 프로젝션 툴입니다.

## 실행

```bash
npm install
npm run dev
```

## 현재 기능

- OBJ 파일 로드
- PNG/JPG/WebP albedo 텍스처 로드
- Three.js 3D 뷰포트와 Orbit 조작
- 좌측 Projection View / 우측 Result View 분리
- UV 기반 Paint/Erase 브러시
- Projection Paint 브러시와 반투명 projection image overlay
- 브러시 size/strength/hardness 조정
- 편집된 albedo PNG export
코덱스 텍스쳐 프로젝션 툴
